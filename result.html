Project Result

The final outcome of our VisionLab website project.

Final Website

Our final result is a structured and responsive website for VisionLab. The website introduces our project, shows our development process, presents our results, and connects visitors to each team member's personal page.

Main Features

  • A clean main page with a hero section and navigation buttons
  • Separate pages for Process, Result, and History
  • Individual profile pages for each team member
  • A GitHub repository button for project access
  • A consistent visual design using HTML and CSS

What We Learned

Through this project, we learned how to structure a website, connect multiple HTML pages, design buttons and cards, and organize project information clearly. We also practiced using GitHub to share our project online.
