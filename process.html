Our Process

How we planned, designed, and built the VisionLab website.

Step 1: Planning the Project

We first decided the main purpose of our website. Our goal was to create a clean and simple project website that introduces VisionLab, our team members, and our project work.

Step 2: Designing the Main Page

We created the main page with a header, navigation buttons, a hero section, an About section, a Project section, and a team introduction section. We focused on making the layout easy to understand and visually organized.

Step 3: Creating Individual Pages

After completing the main page, we created separate pages for each team member. These pages introduce each member's basic information, role, and contribution to the project.

Step 4: Building Project Pages

We added separate pages for the project process, results, and history. This helped us organize our content clearly instead of putting everything on one long page.

Step 5: Uploading to GitHub

Finally, we connected our project to GitHub so that our website and project files could be shared online. This also helped us understand how real web projects are published and managed.
